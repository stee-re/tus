# Tús

> Disclaimer - this is currenty not even feature complete - very WIP

A fast, local-first start page for your browser.

Import your browser bookmarks, organize them into pages and groups, customize icons. On page load, cursor is deliberately placed in the search box so you can get where you want to go without even touching the mouse.

It runs entirely in your browser.  
No server. No telemetry. No accounts.

Everything stays on your machine.

---

## Why Tús?

We all need bookmarks and browsers to a pretty good job of managing all these bookmarks. But its not always easy to quickly find what you need. Its for this reason there are so many Startpage/Dashboard projects out there, aiming to do this. But none (that I could find) offered a simple webapp (with no backed requrements) that allowed me to import the bookmarks I already had. I imagined others might wish for something like that too. So this project aims to be:

- Fast
- Private
- Customizable
- Minimal
- Yours

---

## Features

- Import Chrome & Firefox bookmarks
- Navigate links directly from Search
- Multi-page support
- Edit mode with drag & drop
- Custom icon selection
- Dark mode
- JSON import/export

---

# About the name

Tús is the Irish word for “beginning”.
